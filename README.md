# php-framework-mock
PHP Framework mock

## 仕様

### 必須クラス
- Router
- Logger
- Input & Validation, Output (View)
- Controller [WebController, CliController]

### オプション
- .editorconfig
- PHPUnit

フレームワークに含めたくないけど欲しいやつ
- Swagger
- Docker

### できれば
- Composer からインストール `composer create-project`
- PSR 準拠
